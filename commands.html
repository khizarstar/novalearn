⚙️ Available Commands

/help
General

Displays the complete help menu, available commands, useful links, and guidance for using NOVA CHAT.

/generate
AI

Generate intelligent AI responses, explanations, educational content, and helpful answers instantly.

/activate-channel
Configuration

Sets the current channel as an AI chat channel where users can interact with NOVA CHAT without using commands.

/remove-channel
Configuration

Removes the AI chat functionality from the selected channel and restores it to a normal text channel.

/reset
Configuration

Resets the bot configuration for the current server or selected settings back to their default values.

/ping
Utility

Checks the bot's latency and displays its current response time and connection status.

/vc-join
Voice

Makes NOVA CHAT join your current voice channel for voice-based interactions and future voice features.

/vc-leave
Voice

Disconnects NOVA CHAT from the current voice channel safely.
