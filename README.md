# 🎓 NovaLearn

> An intelligent AI-powered educational Discord bot designed to help students learn, practice, and succeed.

---

## 📖 About

NovaLearn is an educational Discord bot built to make learning simple, interactive, and accessible. It provides instant explanations, homework assistance, exam preparation, and subject-specific guidance for students from **Class 1 to University**.

Whether you're solving mathematics, understanding science, improving your English, or studying Islamic Studies, NovaLearn is available 24/7 to support your learning journey.
